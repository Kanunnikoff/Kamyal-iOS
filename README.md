«Къамаьл» — клавиатура для ввода на ингушском языке в iOS и iPadOS.

## Возможности

- Кириллическая и историческая латинская раскладки.
- Все характерные для ингушского алфавита буквы и буквосочетания.
- Дополнительные символы по долгому нажатию.
- Автоматическая заглавная буква и настраиваемый звук клавиш.
- Подсказки из ингушского частотного словаря для кириллической раскладки.
- Отдельная раскладка эмодзи и интерфейс для iPhone и iPad.
- Русский и ингушский варианты подписей в приложении и на служебных клавишах.

## Как включить клавиатуру

1. Установите и откройте приложение.
2. Перейдите в «Настройки» → «Основные» → «Клавиатура» → «Клавиатуры».
3. Нажмите «Новые клавиатуры» и выберите «Къамаьл».
4. В поле ввода удерживайте кнопку переключения клавиатур и выберите «Къамаьл».

iOS использует системную клавиатуру в полях пароля и номера телефона. Отдельные приложения также могут запрещать сторонние клавиатуры.

## Системные требования

- iOS или iPadOS 16.0 и новее.
- Xcode с поддержкой Swift Package Manager — для сборки из исходного кода.

## Сборка проекта

1. Откройте `Kamyal.xcodeproj` в Xcode.
2. Дождитесь загрузки пакетов KeyboardKit и Firebase.
3. При необходимости выберите свою команду разработчика для целей `Kamyal`, `Keyboard` и `Stickers`.
4. Выберите схему `Kamyal` и соберите приложение для нужного устройства.

Основное приложение и расширение `Keyboard` обмениваются настройками через общую группу приложений. Её идентификатор должен совпадать в возможностях обеих целей и в `Config.APP_GROUP_NAME`.

## Данные словаря

Подсказки формируются на основе ингушского частотного словаря [YandievRuslan/ing_freq_dict](https://github.com/YandievRuslan/ing_freq_dict).
